# BunnyHop

**BunnyHop** is an application designed to help individuals who feel shy or anxious in social situations gradually build confidence and social courage

## Features
1. **Task-based Flashcards:** Provides daily challenges and activities to help users overcome fears and gain confidence  
2. **Guidance & Tips:** Offers motivational messages and practical advice to encourage users in social interactions  
3. **Activity Calendar:** Allows users to record activities and track their feelings to monitor personal growth

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- MIT App Inventor

## Goal
To support introverted or socially anxious users in gradually gaining confidence through structured activities and motivational guidance
